# AI-Career-Mentor
#  AI Career Mentor using Langflow

An AI-powered Career Mentor built using **Langflow**, **Google Gemini 2.5 Flash**, and a **Python Interpreter Tool** to provide personalized career guidance for students.

---

##  Problem Statement

Many students struggle to identify the right skills, projects, certifications, and preparation strategy for placements. They often receive generic advice instead of guidance tailored to their goals.

This AI Career Mentor helps students by providing personalized career advice, learning roadmaps, project ideas, and placement preparation strategies.

---

##  Features

- 🎓 Personalized Career Guidance
- 🛣️ Learning Roadmaps
- 💻 Project Recommendations
- 📜 Certification Suggestions
- 💼 Placement Preparation
- 🧠 Interview Tips
- 🐍 Python Tool Integration for calculations and code execution

---

##  Tech Stack

- Langflow
- Google Gemini 2.5 Flash
- Python Interpreter Tool
- Python

---

##  Langflow Flow

```text
Chat Input
      │
      ▼
Agent (Gemini 2.5 Flash)
      ▲
      │
Python Interpreter Tool
      │
      ▼
Chat Output
```

##  Demo

The AI agent can answer career-related questions such as:

- "I'm a CSE student in my 5th semester. Can you create a roadmap to get a 15 LPA package?"
- "Suggest AI projects for my resume."
- "How should I prepare for campus placements?"

It can also execute Python when needed, for example:

- "Use Python to calculate the factorial of 20."
